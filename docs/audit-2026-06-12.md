## Security scan — confirmed OK

- **Secrets:** config dir 0700 / file 0600 with silent perm-heal (`config.ts:159-163`); no API key reaches console, display, daemon.log, or cost.log. Scrubber covers Anthropic/OpenAI/GitHub/AWS keys, Bearer tokens, emails, absolute paths. Scrub runs BEFORE distill/write/DB `content` (`run.ts:389-425`) — no raw-transcript leak path found.
- **Injection:** every `spawnSync` arg-array (launchd, systemd, cron, mcp/install, osascript with AppleScript escaping); all SQL prepared statements, zero interpolation.
- **Fetch timeouts:** callKie 120s AbortController, embedder 10s + 3s ping, doctor probe 15s race. Anthropic SDK self-handles.
- **Conventions:** zero `process.exit()`; all 22 cli.ts actions wrapped in `runAction`; tool-block grammar owned solely by `toolCallFilter.ts`; no dead exports found.
- **Plugin:** no innerHTML/eval, `spawn()` arg-arrays with SIGKILL timeouts, correct Obsidian lifecycle cleanup, zero console noise.
- **Tarball:** `npm pack --dry-run` → 82 files, 151 kB, dist/ + allowlisted assets only. No personal leaks.

Discrepancy: plugin manifest/package.json read **0.1.3** locally vs 0.1.2 in handoff — verify before next plugin ship.
